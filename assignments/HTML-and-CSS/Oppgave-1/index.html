Information theory

Because info? For information??

The basis of information theory is the information content of the probability P(X = x) of the specific outcome x of a random variable X. It is defined as

I(x) = -logP(X = x)

This can be thought as the surprisal of the outcome x occurring. This is especially clear in the boundary cases: when P(X = x) = 0, the logarithm is undefined, and the information content is infinite; i.e. if an outcome has a zero probability, the surprisal of its occurrence is unbounded; if P(X = x) = 1, the outcome x is guaranteed, and the surprisal of the outcome is nil.

In other words, I(x) quantifies how much information is gained from finding that a random process results in the result X = x.

Assume now that X takes one of several values x ∈ E for a set of values E, each x corresponding to a probability P(X = x), such that each P(X = x) is non-negative, and the sum of P(X = x) for all x is 1. The set { P(X = x) | x ∈ E } then defines a probability distribution. One may then define the average information content of the probability distribution {P(X = x)} the outcomes x, with respect to the probability distribution itself:

⟨I(x)⟩ = -⟨logP(X=x)⟩ = -∑xP(X = x) logP(X = x)

This is also denoted H(X), and is called the Shannon entropy of the random variable X.

Some notes on these files

Headers

It may be noticed in the source code that two types of headers, H2 and H3, are used to headline the different pages here. The reason for this is that some pages might take a H2 header and then go straight to a paragraph of normal text. In that case, an issue arises with the distance between the header and the text. To account for this, H2 and H3 are set to have different margin values in the CSS, so that when H4 is used above the first paragraph, H2 may be used, while with no H4, H3 is used for the header.

It is hoped that this was somewhat coherent an explanation. An illustration can be found by clicking "More still" in the menu.
